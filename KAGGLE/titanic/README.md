# Titanic Survival Prediction

## Overview
Predict passenger survival using machine learning models.

## Dataset
Kaggle Titanic dataset

## Features
- Pclass
- Sex
- Age
- Fare
- Embarked
- Title
- FamilySize

## Models
- Logistic Regression (baseline)
- RandomForestClassifier (final)

## Process
1. EDA
2. Feature Engineering
3. Encoding
4. Model Training

## Result
- Score: **0.83**

## What I Learned
- Importance of feature engineering
- Basic ML pipeline structure
- Model comparison (linear vs tree-based)
